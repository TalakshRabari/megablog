# 📝 MegaBlog

MegaBlog is a modern and responsive blogging platform developed using HTML, CSS, JavaScript, and Tailwind CSS. The project provides a clean user interface for displaying blog posts, reading articles, and managing blog content with a mobile-first responsive design.

## 🚀 Features

### User Features

* Responsive blog homepage
* Dynamic blog listing interface
* Individual blog post pages
* Clean and modern UI design
* Mobile-first responsive layout
* Smooth navigation experience
* Cross-browser compatibility

### Blog Management Features

* Blog submission form
* Client-side form validation
* Dynamic content rendering using JavaScript
* Reusable UI components
* Structured and maintainable code architecture

## 🛠️ Technology Stack

### Frontend

* HTML5
* CSS3
* JavaScript
* Tailwind CSS

## 🔧 Installation

### Clone the Repository

```bash
git clone https://github.com/TalakshRabari/megablog.git
```

### Navigate to Project Directory

```bash
cd megablog
```

### Run the Project

Open the project in your browser:

```bash
index.html
```

Or use VS Code Live Server:

```bash
Right Click → Open with Live Server
```
